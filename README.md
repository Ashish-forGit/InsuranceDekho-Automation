## 📌 Project Overview

This repository contains an end-to-end automation testing framework developed for the **InsuranceDekho** application.

The framework supports:

✅ UI Automation Testing

✅ Data-Driven Testing

✅ Page Object Model (POM)

✅ BDD Automation using Cucumber

✅ Reusable Utility Components

✅ Detailed Test Reporting

✅ Cross Browser Execution

---

## 🎯 Key Objectives

- Automate critical business workflows
- Improve test execution speed
- Reduce manual testing effort
- Increase regression test coverage
- Enable scalable automation development

---

# 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/Ashish-forGit/InsuranceDekho-Automation.git
```

### Navigate to Project

```bash
cd InsuranceDekho-Automation
```

### Install Dependencies

```bash
mvn clean install
```

---

# ▶️ Run Test Cases

## TestNG Framework

```bash
mvn test
```

## BDD Framework

```bash
mvn test -Dcucumber.filter.tags="@Smoke"
```
